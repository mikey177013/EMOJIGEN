## üöÄ Installation

To get EMOJIGEN up and running on your local machine, follow these simple steps.

### Prerequisites

Ensure you have the following installed:

*   [Node.js](https://nodejs.org/en/download/) (LTS version recommended)
*   [npm](https://www.npmjs.com/get-npm) (usually comes with Node.js) or [Yarn](https://classic.yarnpkg.com/en/docs/install/)

### Steps

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/mikey177013/EMOJIGEN.git
    cd EMOJIGEN
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # OR
    yarn install
    ```

3.  **Start the development server**:
    ```bash
    npm start
    # OR
    yarn start
    ```

    This will typically open EMOJIGEN in your default web browser at `http://localhost:3000`.

## üí° Usage

Once the development server is running, you can start creating and customizing emojis!

1.  **Navigate to the application**: Open your web browser and go to `http://localhost:3000`.
2.  **Select a base emoji**: Choose an emoji from the available library to begin customizing.
3.  **Apply customizations**: Use the provided tools and sliders to change colors, add accessories, or alter facial features.
4.  **Preview in real-time**: Observe the changes instantaneously in the preview window.
5.  **Export your creation**: Once satisfied, use the export option to download your custom emoji.
